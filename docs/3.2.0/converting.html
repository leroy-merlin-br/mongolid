Converting models

Converting to arraysâ€‹

When building JSON APIs, you may often need to convert your models to arrays. So, Mongolid includes methods for doing so. To convert a model and its loaded relationship to an array, you may use the toArray method:

Converting a model to an array
    $user = User::first();
    
    return $user->toArray();
tip

Note that cursors can be converted to array too:

    return User::all()->toArray();
